Key Features Shown

  • Bandleader dropdown — select an artist, shows recording count
  • Filter dropdowns — Genre, Singer (populated dynamically per artist)
  • Free-text search — searches across title, composer, author
  • Sortable columns — click headers to sort (arrow indicators)
  • Genre color badges — Tango (red), Vals (blue), Milonga (green)
  • Summary stats bar — genre breakdown and year range
  • Expandable row detail — click a row to see all 18 fields
  • CSV download buttons — current artist or all artists
  • Pagination — 50 rows per page
