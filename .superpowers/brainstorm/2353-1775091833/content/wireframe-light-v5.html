Suggested UX Improvements

Additional refinements worth considering

A

Hover highlight on rows

Subtle background change on mouse hover so users know rows are clickable. Pairs with the selected-row highlight.

B

Active filter chips

When a filter is active, show a small chip/tag below the filter bar (e.g. Genre: Vals ×) so users can see and clear active filters at a glance.

C

Sort direction indicators

Show a small ▲/▼ arrow next to the currently sorted column header. Clicking toggles ascending/descending.

D

Keyboard navigation

Arrow keys to move between rows, Enter to select/deselect, Escape to clear selection. Useful for power users browsing large discographies.

E

Responsive column hiding

On narrower screens, auto-hide Composer and Author columns first, then Grouping. The detail panel always shows all fields regardless.

F

"Back to top" button

Since the table scrolls instead of paginates, show a small floating button to jump back to the top after scrolling deep into a large discography.
